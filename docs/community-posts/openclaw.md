```text
Reel Notes: an OpenClaw-ready tool for turning saved reels into Markdown notes
```

Body:

```text
I published Reel Notes, an open-source repo that converts saved Instagram reels into readable Markdown notes.

It includes an OpenClaw runbook:
https://github.com/gondalaimafia/reel-notes/blob/main/examples/agents/openclaw-runbook.md

The runbook tells OpenClaw to:
- run setup
- verify dependencies
- avoid committing secrets or generated data
- open the manual Instagram login browser
- check auth status
- run a small import before any backlog job

The output works with Obsidian by default, but it can write plain Markdown to any folder.

Article:
https://gondalaimafia.github.io/reel-notes/

Repo:
https://github.com/gondalaimafia/reel-notes
```
