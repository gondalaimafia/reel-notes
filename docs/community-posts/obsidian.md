# Obsidian Community Post

Title:

```text
Reel Notes: turn saved Instagram reels into Obsidian notes
```

Body:

```text
I built an open-source tool for people who save useful Instagram reels and then never find them again.

Reel Notes uses your own browser session, extracts reel audio, transcribes it, asks an LLM to create a topic title and key takeaway lesson, and writes one Markdown note per reel.

It is Obsidian-first:
- frontmatter
- tags
- aliases
- wikilinks
- topic-based filenames
- key takeaway lesson
- reader-friendly notes
- raw transcript kept for traceability

It can also write plain Markdown for any folder-based notes app.

Repo:
https://github.com/gondalaimafia/reel-notes

Article:
https://gondalaimafia.github.io/reel-notes/

The tool does not ask for your Instagram password. You log in manually in a browser profile you control, and the importer reuses that session.
```
